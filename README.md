# Login Application (React + Node.js)

This project is a simple login application built using React for frontend and Node.js (Express) for backend.

## Features
- Login screen with Email & Password
- Basic validation (required fields, email format)
- Backend login API with hardcoded user
- Success and error responses
- No database used

## Tech Stack
- Frontend: React, Tailwind CSS
- Backend: Node.js, Express
- API Communication: Fetch API
